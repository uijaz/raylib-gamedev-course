# Contributing to raylib-gamedev-course

Thank you for your interest in contributing! This course is built for the community, by the community.

---

## 🎯 How You Can Help

### Priority Areas

| Area | Description |
|------|-------------|
| **Linux Setup Guide** | Step-by-step instructions for Ubuntu/Debian and Fedora |
| **Windows Setup Guide** | Step-by-step instructions for MSYS2/MinGW |
| **Module Content** | Improve explanations, add examples, fix typos |
| **Stretch Goals** | Add more challenging exercises to existing modules |
| **Testing** | Test modules on different operating systems |
| **Translations** | Translate READMEs to other languages |

---

## 📋 How to Contribute

### 1. Open an Issue

Before making changes, open an issue to discuss what you'd like to work on. This ensures no duplicate work and aligns with the course goals.

### 2. Fork the Repository

Click the "Fork" button on GitHub to create your own copy.

### 3. Create a Branch

```bash
git checkout -b feature/your-feature-name
```

### 4. Make Your Changes

- Follow the existing style and structure
- Keep code simple and beginner-friendly
- Test your changes locally

### 5. Commit and Push

```bash
git commit -m "Add your change description"
git push origin feature/your-feature-name
```

### 6. Open a Pull Request

Go to the original repository and click "New Pull Request". Describe your changes clearly.

## 📝 Style Guidelines

### README Files

- Use clear, simple language
- Include code blocks with proper syntax highlighting
- Follow the existing section structure
- Add emojis for visual cues (🎯, 📘, 🧱, etc.)

### Code

- Use consistent indentation (4 spaces)
- Comment complex logic
- Follow the `Game` struct pattern
- Keep functions small and focused

### Build Commands

- Include commands for macOS, Linux, and Windows
- Test on at least one platform before submitting

---

## 🧪 Testing

If adding a new module or modifying existing ones, please test:

- Code compiles without warnings
- Game runs as expected
- Build commands work on your platform
